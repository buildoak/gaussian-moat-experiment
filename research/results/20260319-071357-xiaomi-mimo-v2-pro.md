### 2.3 Percolation Theory Framework

The Gaussian prime graph resembles **continuum percolation** with site density ρ(N) ~ 4/(π ln N).

Critical phenomena: For random geometric graphs, percolation occurs when average degree exceeds a threshold. Here, the "average degree" at radius r is:
- Number of primes in annulus ~ 2πr·(dr)·ρ(r²) ~ (8r dr)/(π ln r²)
- Connectivity radius = √40 ≈ 6.325
- Expected neighbors of a prime at distance r ~ π(√40)²·ρ(r²) = 40·4/(π ln r²) = 160/(π ln r²)

For percolation to persist to infinity, we need expected degree > critical value d_c (≈ 1.44 for 2D Poisson blob model).
Set 160/(π ln r²) > 1.44 → ln r² < 160/(1.44π) ≈ 35.4 → r² < e³⁵·⁴ ≈ 2.4 × 10¹⁵ → r < 1.5 × 10⁷

This gives an **upper bound** of ~15 million, but it's crude because:
1. Primes aren't Poisson—they're deterministic with multiplicative structure
2. The graph isn't random geometric—it's deterministic with specific lattice constraints
3. The bound assumes infinite connectivity requires positive density, but moats can form even with positive density if there are "holes" in the prime distribution

### 2.4 Prime Distribution & Gap Analysis

Gaussian primes in first octant: count ~ x²/(4 ln x) for |z| ≤ x.

The probability that a random Gaussian integer near N is prime ~ 4/(π ln N).

For a "moat" of width √40 to exist, we need an annular region where **no two primes within √40 are connected**. This requires either:
1. **Local depletion:** A region with no primes at all (unlikely for width √40)
2. **Connectivity failure:** Primes exist but are isolated or form disconnected clusters

Given that at k²=32, 75% of all primes are in the connected component, complete depletion is unlikely. The moat likely forms via **fragmentation**—the component breaks into finite pieces while primes remain locally dense.

### 2.5 Branching Process Analogy

View exploration from origin as a branching process:
- Each prime has ~18 possible directions (p(40)=18)
- But only ~ρ(N) fraction of neighbors are actually prime
- Effective branching factor b(N) ~ 18·ρ(N) ~ 18·4/(π ln N) = 72/(π ln N)

For survival to distance R, we need b(N) > 1 on average along the path.
Set 72/(π ln R²) > 1 → ln R² < 72/π ≈ 22.9 → R < e¹¹·⁴⁵ ≈ 94,000

This gives ~94,000, far below observed values. The branching process model is too pessimistic because:
1. It assumes independence (primes aren't random)
2. The component can survive with b(N) < 1 temporarily if it finds "fertile" regions
3. The process isn't memoryless—path history matters

## 3. Mechanism of Moat Formation

**Assessment:** Moats form primarily via **connected component fragmentation**, not local prime depletion.

Evidence:
1. At k²=32, component contains 75% of all primes—density isn't the issue
2. Upper bounds for k²=34,36 are finite but large, suggesting the component is huge before termination
3. The deceleration in growth suggests diminishing returns as the component expands—it becomes harder to find new connections

**Proposed mechanism:** As the component grows, it encounters regions where primes are arranged in "islands" separated by gaps > √40. These gaps aren't devoid of primes, but the primes are arranged such that no path can bridge them with steps ≤ √40. This is a **geometric connectivity** issue, not a density issue.

The "origin-lineage extinction" hypothesis is plausible: the component might die out not because it hits a prime desert, but because all branches of the exploration tree simultaneously hit impassable gaps.
