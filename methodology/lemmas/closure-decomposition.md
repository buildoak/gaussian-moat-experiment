# Closure Decomposition Lemma

**Role:** Supportive lemma for TileOp legitimacy  
**Proves:** If the connectivity operator is a closure operator and the partition is edge-complete, the abstract decomposition (compute locally, merge, close) recovers global components. Bridges to the real pipeline require a separate port fidelity argument.

---

## Acting Set

V — finite set of vertices. E — set of unordered pairs {u, v} from V (u != v). Graph G = (V, E).

For Gaussian moat: V = Gaussian primes in a bounded region, {p, q} in E iff ||p - q||^2 <= K. The results below are general.

## Union-Find as Closure Operator

cl operates on symmetric relations over V. Define cl(E) — the equivalence relation on V generated by edge set E. Concretely: u ~ v under cl(E) iff a path u = w_0, w_1, ..., w_m = v exists with each {w_j, w_{j+1}} in E. Connected components = equivalence classes of cl(E).

Union-Find is an algorithm that computes cl(E). Output depends only on E, not on processing order.

**cl is a closure operator.** Three properties, each immediate:

1. **Extensive:** E ⊆ cl(E) — every edge is a length-1 path.
2. **Monotone:** E_1 ⊆ E_2 implies cl(E_1) ⊆ cl(E_2) — more edges, more paths.
3. **Idempotent:** cl(cl(E)) = cl(E) — transitive closure of an equivalence relation is itself.

## Decomposition Theorem

**For any closure operator cl:**

> cl(A_1 ∪ ... ∪ A_n) = cl( cl(A_1) ∪ ... ∪ cl(A_n) )

**Proof.**

(⊆) A_i ⊆ cl(A_i) by extensivity. So ∪A_i ⊆ ∪cl(A_i). So cl(∪A_i) ⊆ cl(∪cl(A_i)) by monotonicity.

(⊇) A_i ⊆ ∪A_i, so cl(A_i) ⊆ cl(∪A_i) by monotonicity, so ∪cl(A_i) ⊆ cl(∪A_i). Then cl(∪cl(A_i)) ⊆ cl(cl(∪A_i)) = cl(∪A_i) by monotonicity + idempotence. ∎

## Application to Tiled UF

Let {V_1, ..., V_n} be subsets of V. Each induces E_i = { {u,v} in E : u, v in V_i }.

**Edge completeness:** E_1 ∪ ... ∪ E_n = E (no edge lost).

If edge completeness holds, the theorem gives:

> cl(E) = cl( cl(E_1) ∪ ... ∪ cl(E_n) )

Compute components within each subset independently, merge with one final closure step — equals global components.

**Abstract merge step:** initialize fresh UF over V, for each subset union all vertices sharing a local component. This implements the outer cl of the theorem. Note: the real tiling pipeline does not pass full cl(E_i) between tiles — it passes a compressed boundary encoding (ports/groups). Proving that this encoding faithfully represents cl(E_i) at tile boundaries is a separate claim (port fidelity).

## Conditions for the Abstract Decomposition

Two conditions make the decomposition theorem applicable:

1. **Closure operator** — the connectivity operator must satisfy extensivity, monotonicity, idempotence. UF / transitive closure does. Verified above. This is structural — it holds by definition.
2. **Edge completeness** — the partition must satisfy ∪E_i = E. This is a condition on the tiling geometry, verified separately per tiling scheme.

Edge completeness is necessary: if some edge {u, v} is missing from every E_i, the graph E = {{u, v}} is a counterexample — tiled processing cannot connect u and v. The closure operator property is the algebraic foundation that makes edge completeness sufficient.

Together these two conditions are sufficient for the abstract decomposition. For the full pipeline (where tiles communicate via compressed encodings rather than full cl(E_i)), a third condition — port fidelity — is additionally required.
